# Fantasy F1 Picks MVP Checklist

## Phase 1
- Scaffold Vite + React + TypeScript app
- Add Tailwind, routing, forms, charts, Supabase client, and test tooling
- Define Supabase schema and RLS strategy
- Define route map, app shell, and shared UI primitives

## Phase 2
- Implement email allowlist + shared password auth flow
- Build dashboard, submit picks, all picks, results, and admin screens
- Add lock-time and visibility enforcement in shared business logic

## Phase 3
- Add scoring recalculation flow and stats widgets
- Cover critical logic with tests
- Run lint, typecheck, tests, and document setup/deployment
